# Production Backend Tasks

Starting code for the Preparing for Deployment chapter in TripleTen's AI Software Engineering program.

## Setup

```bash
npm install
```

The backend runs on `http://localhost:3000`. The frontend (used only in the CORS lesson) can be run with Live Server.

## Tests

Tests can be run via the provided `npm` scripts.

```bash
npm run test:01
```

Replace `02` with the lesson number.

## Solutions

Solutions are found in the `solutions` branch.
